# 安全策略

`eBPFDexDumper-rs` 面向授权的 Android 逆向分析和安全研究场景。请不要把它用于你
没有权限检查的设备、应用或数据。

## 报告安全问题

如果你发现项目本身存在安全问题，请优先私下联系维护者，不要直接在公开 Issue 中
发布可复现利用细节。

报告时建议包含：

- 受影响版本或 commit。
- 问题触发条件。
- 影响范围。
- 最小复现步骤。
- 你认为合适的修复建议。

## 支持范围

维护者会优先处理项目代码、构建脚本、Release 产物和文档中的安全问题。

不属于本项目安全问题的情况包括：

- 未授权使用本工具造成的后果。
- 目标设备、目标应用或第三方系统自身的问题。
- 因 root 环境、SELinux 配置、内核限制导致的运行失败。
